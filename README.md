# The Last Cabin
This is the remake of the Original "Figma Make" game, built with Meta Horizon and TypeScript

This is a code bundle for The Last Cabin. The original project is available at https://www.figma.com/design/ZBWGawHh65D6Q5qCFm0ZO3/The-Last-Cabin.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
